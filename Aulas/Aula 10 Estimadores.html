1. Fundamentos Tericos: Funes de Estimao

[cite_start]Na estatstica, os estimadores clssicos podem ser obtidos de forma unificada por meio de Funes de Estimao[cite: 545]. [cite_start]Este mtodo generaliza abordagens como a Mxima Verossimilhana e o Mtodo de Momentos[cite: 546].

Definio Cannica (Definio 2.4.3)

Seja $Z_1, ..., Z_n$ uma sequncia de vetores aleatrios. Uma funo de estimao no-viesada para o parmetro $\theta \in \Theta \subseteq \mathbb{R}^p$ definida como:

$$U_n(\theta) = \frac{1}{n}\sum_{k=1}^{n}u_k(Z_k, \theta)$$

[cite_start]Esta funo deve satisfazer a condio de no-vis[cite: 554], ou seja, a esperana de cada componente da soma deve ser zero para o verdadeiro valor do parmetro:

$$E_{P_{\theta}}(u_k(Z_k, \theta)) = 0, \quad \forall \theta \in \Theta$$

O estimador $\hat{\theta}_n$ ento definido como a raiz da funo de estimao, ou seja, o valor que a anula:

$$U_n(\hat{\theta}_n) = 0$$

2. O Mtodo de Momentos (EMM) como Funo de Estimao

[cite_start]O Mtodo de Momentos (EMM) consiste em igualar os momentos populacionais aos seus anlogos amostrais para encontrar estimativas para os parmetros[cite: 591]. Ele um caso particular do mtodo das funes de estimao.

Para o j-simo momento, a funo $u_k$ definida como a diferena entre a varivel aleatria elevada potncia $j$ e o seu respectivo momento populacional:

$$u_k(Z_k, \theta) = Z_k^j - h_j(\theta), \quad \text{onde } h_j(\theta) = E_{\theta}(Z^j)$$

Estudo de Caso: A Distribuio de Poisson

Vamos considerar uma amostra aleatria $Z_1, ..., Z_n$ de uma populao $Z \sim \text{Pois}(\theta)$, onde $\theta > 0$. Para esta distribuio, os dois primeiros momentos populacionais so:

  1. Primeiro Momento: $E_{\theta}(Z) = \theta$
  2. Segundo Momento: $E_{\theta}(Z^2) = \text{Var}_{\theta}(Z) + [E_{\theta}(Z)]^2 = \theta + \theta^2$

A partir desses momentos, podemos construir dois estimadores diferentes para $\theta$.

Estimador 1: EMM1 (baseado no 1 momento)

A funo de estimao : $U_{n,1}(\theta) = \frac{1}{n}\sum_{k=1}^{n}(Z_k - \theta)$

Resolvendo $U_{n,1}(\hat{\theta}_1) = 0$, obtemos: $\frac{1}{n}\sum Z_k - \hat{\theta}_1 = 0 \implies \hat{\theta}_1 = \overline{Z}$

[cite_start]Este estimador, a mdia amostral, tambm o Estimador de Mxima Verossimilhana (EMV) para o parmetro da Poisson, sendo conhecido por suas propriedades de eficincia assinttica[cite: 586, 614].

Estimador 2: EMM2 (baseado no 2 momento)

A funo de estimao : $U_{n,2}(\theta) = \frac{1}{n}\sum_{k=1}^{n}(Z_k^2 - (\theta + \theta^2))$

Resolvendo $U_{n,2}(\hat{\theta}_2) = 0$, obtemos: $\frac{1}{n}\sum Z_k^2 - (\hat{\theta}_2 + \hat{\theta}_2^2) = 0 \implies \hat{\theta}_2^2 + \hat{\theta}_2 - \overline{Z^2} = 0$

Esta uma equao quadrtica em $\hat{\theta}_2$. Usando a frmula de Bhaskara e sabendo que $\theta > 0$, pegamos a raiz positiva: $$\hat{\theta}_2 = \frac{-1 + \sqrt{1 + 4\overline{Z^2}}}{2}$$

Temos agora dois estimadores diferentes para o mesmo parmetro $\theta$. Qual deles melhor?

3. Comparando Estimadores: O Erro Quadrtico Mdio (EQM)

[cite_start]A qualidade de um estimador pode ser medida pelo seu Erro Quadrtico Mdio (EQM)[cite: 690]. Para um estimador $T(Z_n)$ de uma quantidade $g(\theta)$, o EQM definido como:

$$EQM_{\theta}(T(Z_n), g(\theta)) = E_{\theta}[(T(Z_n) - g(\theta))^2]$$

[cite_start]Um estimador com menor EQM prefervel, pois ele est, em mdia, mais prximo do valor que se deseja estimar[cite: 660]. [cite_start]O EQM combina a varincia do estimador e o seu vis ao quadrado[cite: 692].

Vamos usar uma simulao de Monte Carlo para estimar o EQM de $\hat{\theta}_1$ e $\hat{\theta}_2$ e determinar qual o melhor na prtica.

4. Simulao de Monte Carlo em R

O objetivo da simulao gerar muitas amostras de uma populao de Poisson com um $\theta$ conhecido (que chamaremos de theta0), calcular as duas estimativas ($\hat{\theta}_1$ e $\hat{\theta}_2$) para cada amostra, e ento calcular a mdia dos erros quadrados para estimar o EQM.

In[1]:
# Definio dos parmetros da simulao
set.seed(42)  # Garante a reprodutibilidade dos resultados
n = 100       # Tamanho amostral
M = 10000     # Nmero de rplicas de Monte Carlo
theta0 = 10   # O verdadeiro valor de theta

# Vetores para armazenar as estimativas
theta1_vetor = numeric(M)
theta2_vetor = numeric(M)

# 3. Loop de Simulao
for (i in 1:M) {
  # Gerar uma amostra de uma populao Pois(theta0) de tamanho n
  z = rpois(n, theta0)
  
  # Calcular e armazenar a estimativa de theta1 (EMM1 / EMV)
  theta1_vetor[i] = mean(z)
  
  # Calcular e armazenar a estimativa de theta2 (EMM2)
  media_z_quadrado = mean(z^2)
  theta2_vetor[i] = (sqrt(4 * media_z_quadrado + 1) - 1) / 2
}

# 4. Clculo do EQM estimado para cada estimador
# EQM  a mdia dos erros quadrados
EQM1 = mean((theta1_vetor - theta0)^2)
EQM2 = mean((theta2_vetor - theta0)^2)

# 5. Apresentao dos resultados
cat("Resultado da Simulao de Monte Carlo\n")
cat("---------------------------------------\n")
cat("Tamanho da Amostra (n):", n, "\n")
cat("Nmero de Rplicas (M):", M, "\n")
cat("Valor Verdadeiro (theta0):", theta0, "\n\n")

6. Anlise Assinttica e Verificao do Teorema do Limite Central (TLC)

Na seo anterior, vimos empiricamente que $\hat{\theta}_1$ teve um EQM menor que $\hat{\theta}_2$ para $n=100$. A anlise assinttica nos ajuda a entender o comportamento desses estimadores quando o tamanho da amostra $n \to \infty$. O Teorema do Limite Central para Estimadores (Teorema 2.4.1) a ferramenta fundamental para isso.

Teoria: A Distribuio Assinttica dos Estimadores

Conforme suas anotaes, podemos aplicar o TLC para encontrar a distribuio assinttica de ambos os estimadores.

Estimador 1: $\hat{\theta}_{n,1} = \overline{Z}$

Para o primeiro estimador, o TLC clssico se aplica diretamente: $$\sqrt{n}(\hat{\theta}_{n,1} - \theta) = \sqrt{n}(\overline{Z} - \theta) \xrightarrow{D} N_1(0, \text{Var}_{\theta}(Z))$$ Como para a Poisson temos $\text{Var}_{\theta}(Z) = \theta$, o resultado : $$\sqrt{n}(\hat{\theta}_{n,1} - \theta) \xrightarrow{D} N_1(0, \theta)$$

Estimador 2: $\hat{\theta}_{n,2}$

Para o segundo estimador, a derivao da varincia assinttica mais complexa. Ela envolve o clculo da varincia de $Z^2$, que por sua vez requer o 3 e 4 momentos da Poisson. Como calculado em suas anotaes, a varincia de $Z^2$ $\text{Var}_{\theta}(Z^2) = 4\theta^3 + 6\theta^2 + \theta$. A aplicao do Mtodo Delta leva ao resultado: $$\sqrt{n}(\hat{\theta}_{n,2} - \theta) \xrightarrow{D} N_1\left(0, \frac{4\theta^3+6\theta^2+\theta}{(1+2\theta)^2}\right)$$

A varincia assinttica do EMV/EMM1, que $\theta$, a menor varincia possvel (atinge o limite de Cramr-Rao). Isso explica teoricamente por que ele mais eficiente e por que observamos um EQM menor na simulao.

Simulao: Verificando o TLC

O objetivo agora usar uma simulao de Monte Carlo para verificar visualmente se as distribuies dos estimadores, quando devidamente padronizados, se aproximam de uma Normal Padro $N(0,1)$, como prev a teoria.

Para padronizar, usamos a frmula: $$Z_{norm} = \frac{\sqrt{n}(\hat{\theta}_n - \theta_0)}{\sqrt{\text{Varincia Assinttica Terica}}}$$

REVISO

1. Os Princpios Distintos de EMV e EMM

Como discutido, o Estimador de Mxima Verossimilhana (EMV) e o Estimador de Momentos (EMM) so derivados de princpios fundamentalmente diferentes:

  • EMV (Mxima Verossimilhana): Busca o valor do parmetro que maximiza a probabilidade (ou densidade) da amostra observada. Ele utiliza a informao completa da distribuio contida na funo de verossimilhana. Sua filosofia : "Qual valor de $\theta$ torna os dados que eu observei os mais provveis?".

  • EMM (Momentos): Busca o valor do parmetro que iguala os momentos tericos da populao (como a mdia $E[X]$) aos seus correspondentes empricos na amostra (como a mdia amostral $\overline{X}$). Ele utiliza uma informao parcial da distribuio. Sua filosofia : "Qual valor de $\theta$ faz com que a mdia terica do modelo seja igual mdia que observei?".

Embora possam coincidir em certos casos (como na Poisson), eles no so, em geral, o mesmo. Um no est englobado no outro. Para evidenciar essa diferena, vamos analisar um caso em que eles produzem resultados distintos.

2. Estudo de Caso: A Distribuio Uniforme $U(0, \theta)$

Seja $Z_1, ..., Z_n$ uma amostra aleatria de uma populao com distribuio Uniforme no intervalo $(0, \theta)$. A funo densidade de probabilidade (f.d.p.) :

$$f(z|\theta) = \frac{1}{\theta}, \quad \text{para } 0 < z < \theta$$

Vamos derivar os dois estimadores para $\theta$.

Derivao do Estimador de Momentos (EMM)

  1. Primeiro Momento Populacional: A esperana de uma $U(0, \theta)$ $E_{\theta}(Z) = \frac{0 + \theta}{2} = \frac{\theta}{2}$.
  2. Primeiro Momento Amostral: a mdia amostral, $\overline{Z}$.
  3. Igualando os Momentos: Igualamos o momento populacional ao amostral e resolvemos para $\theta$: $$\frac{\hat{\theta}_{MM}}{2} = \overline{Z} \implies \hat{\theta}_{MM} = 2\overline{Z}$$

Derivao do Estimador de Mxima Verossimilhana (EMV)

  1. Funo de Verossimilhana: A verossimilhana da amostra $z_n$ o produto das densidades: $$L(\theta|z_n) = \prod_{i=1}^{n} \frac{1}{\theta} = \left(\frac{1}{\theta}\right)^n$$ Esta funo s vlida se todas as observaes $z_i$ estiverem no intervalo $(0, \theta)$, o que significa que $\theta$ deve ser maior ou igual ao mximo valor observado na amostra, $Z_{(n)} = \max(Z_1, ..., Z_n)$.
  2. Maximizao: A funo $L(\theta) = (1/\theta)^n$ decrescente em $\theta$. Para maximiz-la, precisamos escolher o menor valor possvel para $\theta$ que ainda seja consistente com os dados. O menor valor que $\theta$ pode assumir o mximo da amostra. $$\hat{\theta}_{MV} = Z_{(n)}$$

Claramente, $\hat{\theta}_{MM} = 2\overline{Z}$ e $\hat{\theta}_{MV} = Z_{(n)}$ so estimadores diferentes. Qual melhor? Vamos usar uma simulao para comparar seus Erros Quadrticos Mdios (EQM).

3. Simulao de Monte Carlo em R para Comparao

Vamos simular o processo para ver qual estimador se comporta melhor na prtica.

In[6]:
# 1. Parmetros da simulao
set.seed(123)
n = 30       # Tamanho da amostra
M = 20000    # Nmero de rplicas
theta0 = 10  # Valor verdadeiro de theta

# 2. Vetores para armazenar as estimativas
theta_mv_vetor = numeric(M)
theta_mm_vetor = numeric(M)

# 3. Loop de Simulao
for (j in 1:M) {
  # Gerar uma amostra de uma populao U(0, theta0)
  amostra = runif(n, min = 0, max = theta0)
  
  # Calcular e armazenar a estimativa de MV
  theta_mv_vetor[j] = max(amostra)
  
  # Calcular e armazenar a estimativa de Momentos
  theta_mm_vetor[j] = 2 * mean(amostra)
}

# 4. Clculo do EQM estimado para cada estimador
EQM_mv = mean((theta_mv_vetor - theta0)^2)
EQM_mm = mean((theta_mm_vetor - theta0)^2)

# 5. Apresentao dos resultados
cat("Resultado da Simulao de Monte Carlo para U(0, theta)\n")
cat("------------------------------------------------------\n")

1. Introduo: Funes de Estimao

Na estatstica clssica, diversos mtodos de estimao podem ser vistos atravs de uma lente unificada: o mtodo das Funes de Estimao. Este framework generaliza tanto o Mtodo de Mxima Verossimilhana (EMV) quanto o Mtodo de Momentos (EMM).

Definio Cannica (Def. 2.4.3)

[cite_start]Uma funo de estimao no-viesada para um parmetro $\theta$ definida como $U_n(\theta) = \frac{1}{n}\sum_{k=1}^{n}u_k(Z_k, \theta)$, onde a condio fundamental que a esperana de cada termo seja nula, i.e., $E_{P_{\theta}}(u_k(Z_k, \theta)) = 0$[cite: 551, 553, 554]. [cite_start]O estimador $\hat{\theta}_n$ ento a raiz que anula esta funo: $U_n(\hat{\theta}_n) = 0$[cite: 566].

2. EMV vs. EMM: Princpios Distintos

Apesar de poderem ser expressos como funes de estimao, o EMV e o EMM surgem de princpios diferentes.

  • [cite_start]EMV (Mxima Verossimilhana): Busca o $\theta$ que maximiza a funo de verossimilhana $L_{z_n}(\theta)$[cite: 604, 638], ou seja, o valor do parmetro que torna a amostra observada o mais provvel possvel. Ele utiliza a informao completa da distribuio.
  • [cite_start]EMM (Momentos): Busca o $\theta$ que iguala os momentos tericos da populao (e.g., $E_\theta(Z)$) aos seus anlogos amostrais (e.g., $\overline{Z}$)[cite: 591]. Ele utiliza uma informao parcial do modelo.

2.1 O Caso da Coincidncia: Famlia Exponencial

Para certas distribuies, como as da famlia exponencial de um parmetro, o EMV e o EMM de primeira ordem podem coincidir.

Demonstrao para a Poisson($\theta$)

Seja $Z_1, ..., Z_n$ uma amostra aleatria de $Z \sim \text{Pois}(\theta)$.

  • EMV: A funo de log-verossimilhana $l(\theta|z_n) = -n\theta + (\sum z_i) \ln(\theta) - \text{const}$. Derivando e igualando a zero: $$\frac{\partial l}{\partial \theta} = -n + \frac{\sum z_i}{\theta} = 0 \implies \hat{\theta}_{MV} = \frac{\sum z_i}{n} = \overline{Z}$$
  • [cite_start]EMM de 1 Ordem: O primeiro momento populacional $E_{\theta}(Z) = \theta$[cite: 731]. Igualando ao momento amostral, temos: $$\hat{\theta}_{MM1} = \overline{Z}$$
  • Concluso: Para a Poisson, $\hat{\theta}_{MV} = \hat{\theta}_{MM1}$.

Demonstrao para a Bernoulli($\theta$)

Seja $Z_1, ..., Z_n$ uma amostra aleatria de $Z \sim \text{Ber}(\theta)$.

  • EMV: A log-verossimilhana $l(\theta|z_n) = (\sum z_i)\ln(\theta) + (n-\sum z_i)\ln(1-\theta)$. Derivando e igualando a zero: $$\frac{\partial l}{\partial \theta} = \frac{\sum z_i}{\theta} - \frac{n-\sum z_i}{1-\theta} = 0 \implies \hat{\theta}_{MV} = \frac{\sum z_i}{n} = \overline{Z}$$
  • [cite_start]EMM de 1 Ordem: O primeiro momento populacional $E_{\theta}(Z) = \theta$[cite: 731]. Igualando ao momento amostral: $$\hat{\theta}_{MM1} = \overline{Z}$$
  • Concluso: Para a Bernoulli, os estimadores tambm coincidem.

2.2 O Caso da Divergncia: Uniforme $U(0, \theta)$

Para mostrar que no so a mesma classe de estimadores, consideramos o caso de $Z \sim U(0, \theta)$.

  • EMV: A verossimilhana $L(\theta) = (1/\theta)^n$ maximizada quando $\theta$ o menor valor possvel, que o mximo da amostra: $\hat{\theta}_{MV} = Z_{(n)}$.
  • EMM de 1 Ordem: O momento populacional $E_{\theta}(Z) = \theta/2$ igualado mdia amostral $\overline{Z}$, resultando em $\hat{\theta}_{MM} = 2\overline{Z}$.

Aqui, os estimadores so claramente diferentes. Vamos usar uma simulao para comparar seu desempenho.

3. Anlise Assinttica e Verificao Prtica

[cite_start]O Teorema do Limite Central para Estimadores (Teorema 2.4.1) nos d a distribuio assinttica de um estimador[cite: 565]. Para um estimador consistente $\hat{\theta}_n$, temos:

$$\sqrt{n}(\hat{\theta}_{n}-\theta)\xrightarrow{D} N(0, B(\theta))$$

Para a inferncia prtica, precisamos de uma quantidade pivotal, cuja distribuio no dependa de parmetros desconhecidos. [cite_start]Pelo Teorema de Slutsky, podemos substituir a varincia assinttica terica $B(\theta)$ por uma estimativa consistente $A_n(\hat{\theta}_n)$[cite: 527, 528], obtendo:

$$\sqrt{n}A_{n}(\hat{\theta}_{n})^{-1/2}(\hat{\theta}_{n}-\theta)\xrightarrow{D} N(0,I)$$
